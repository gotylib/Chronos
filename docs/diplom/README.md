# Диплом в LaTeX

В этой папке лежит самостоятельный проект дипломной работы по теме Chronos.

## Состав

- `main.tex` — главный файл.
- `chapters/` — основные главы.
- `appendix/` — приложения.
- `references.bib` — библиография.

## Сборка

Рекомендуемый способ (XeLaTeX + Biber):

1. `xelatex main.tex`
2. `biber main`
3. `xelatex main.tex`
4. `xelatex main.tex`

Или через `latexmk`:

`latexmk -xelatex -bibtex main.tex`

## Важно

- Текст подготовлен как готовая база диплома по проекту Chronos.
- Под конкретную кафедру обычно нужно:
  - заменить титульный лист на официальный шаблон;
  - добавить индивидуальные требования по оформлению;
  - при необходимости расширить обзор литературы и экспериментальную часть.
