# Chronos.Agent

**HTTP-сервис на машине с Docker**: хранит проекты compose на диске, выполняет `docker compose` по запросам, отдаёт файлы и статус, ведёт метаданные (EF Core: PostgreSQL или SQLite).

## Точка входа

- **`Program.cs`** — конфигурация из переменных окружения, регистрация Minimal API, клиент регистрации в Master, hosted-сервисы (планировщик и др.).

## Назначение папок

| Папка | Роль |
|--------|------|
| `Api/` | Маршруты REST: проекты, compose, жизненный цикл, логи, volumes, диагностика. |
| `Application/` | Прикладная логика поверх домена и инфраструктуры. |
| `Domain/` | Сущности и правила предметной области агента. |
| `Infrastructure/Persistence` | EF DbContext, миграции, доступ к метаданным архивов и заданиям. |

См. также корневой `README.md` и `docs/ARCHITECTURE.md` для связки с Master и стека Docker.
